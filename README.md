# xurl: tool to extract urls from an apk

## install
```bash
➜  ~ cd /opt && git clone https://github.com/b4sh0xf/xurl.git
➜  ~ cd /opt/xurl && chmod +x install.sh
➜  xurl ./install.sh
➜  ~ xurl
```
